💡 Caffeine es una biblioteca de caching en memoria ultra rápida, diseñada para aplicaciones modernas con alta demanda.

🚀 ¿Qué es Caffeine?

Caffeine es una implementación de caché en memoria para Java que reemplaza a Guava y ofrece:

  • Políticas como LRU, LFU.
  • TTL configurable.
  • Concurrencia optimizada.

🔧 Agrega las dependencias en pom.xml

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-cache</artifactId>
</dependency>

<dependency>
    <groupId>com.github.ben-manes.caffeine</groupId>
    <artifactId>caffeine</artifactId>
</dependency>

⚙️ Configuración del cache manager
@Configuration
public class CacheConfig {
    @Bean
    public CacheManager cacheManager() {
        CaffeineCacheManager cacheManager = new CaffeineCacheManager("usuarios", "productos");
        cacheManager.setCaffeine(Caffeine.newBuilder()
                .expireAfterWrite(10, TimeUnit.MINUTES)
                .maximumSize(100));
        return cacheManager;
    }
}

🧪 Uso en un servicio
@Cacheable("usuarios")
public Usuario obtenerUsuario(String id) {
    return usuarioRepository.findById(id).orElseThrow();
}

🧼 Limpieza del cache
@CacheEvict(value = "usuarios", allEntries = true)
public void limpiarCacheUsuarios() {
    // se limpia el cache de usuarios
}

🎯 Conclusión
Caffeine es ideal para mejorar el rendimiento de tu microservicio sin complicarte con soluciones externas.
